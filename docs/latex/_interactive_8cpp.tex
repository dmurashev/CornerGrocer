The \doxylink{class_interactive}{Interactive} class manages the text-\/based user interface for the Corner Grocer system. It displays menu options, validates user input, and calls functions from Inventory\+Manager to load data, print reports, and save backup files.

The menu provides the following options\+:
\begin{DoxyEnumerate}
\item Search for a specific item and display its frequency.
\item Display all item frequencies.
\item Display a histogram of purchases.
\item Exit the program.
\end{DoxyEnumerate}

Input validation ensures that the program handles non-\/numeric input gracefully and avoids infinite loops caused by invalid data.
