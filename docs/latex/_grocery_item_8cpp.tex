Each \doxylink{class_grocery_item}{Grocery\+Item} object stores an item\textquotesingle{}s name and quantity. It provides simple member functions to access and modify this data. The class supports incrementing or decrementing item quantity, as well as retrieving the name and current count.

This class is intentionally lightweight and contains no file I/O or console logic, ensuring that it can be reused in other applications or systems.
