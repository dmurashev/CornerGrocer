\doxysubsection{Detailed Description}
Main entry point for the Corner Grocer item-\/tracking program. 

\begin{DoxyAuthor}{Author}
Dmitrii Murashev 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025-\/10-\/18 
\end{DoxyDate}
This program reads a list of purchased grocery items from an input file, calculates how many times each item appears, and provides a simple text-\/based menu for interacting with this data.

Functionalities include\+:
\begin{DoxyItemize}
\item Searching for a specific item and displaying its purchase frequency.
\item Listing all items and their corresponding frequencies.
\item Displaying a histogram that visually represents item purchase counts.
\item Saving results to a backup file named "{}frequency.\+dat"{}.
\end{DoxyItemize}

The program uses\+:
\begin{DoxyItemize}
\item A {\ttfamily \doxylink{class_grocery_item}{Grocery\+Item}} class to encapsulate each item\textquotesingle{}s name and quantity.
\item An {\ttfamily \doxylink{class_inventory}{Inventory}} class to manage file I/O and data loading.
\item An {\ttfamily \doxylink{class_interactive}{Interactive}} class to handle user interaction and input validation.
\end{DoxyItemize}

All dynamically allocated memory is properly released before exit. Input validation ensures that the user cannot crash the program by entering non-\/numeric data in the menu.
