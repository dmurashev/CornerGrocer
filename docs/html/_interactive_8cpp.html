Detailed Description

Implements the Interactive class responsible for user interface and interaction.

Author
Dmitrii Murashev
Date
2025-10-18

The Interactive class manages the text-based user interface for the Corner Grocer system. It displays menu options, validates user input, and calls functions from InventoryManager to load data, print reports, and save backup files.

The menu provides the following options:

  1. Search for a specific item and display its frequency.
  2. Display all item frequencies.
  3. Display a histogram of purchases.
  4. Exit the program.

Input validation ensures that the program handles non-numeric input gracefully and avoids infinite loops caused by invalid data.
